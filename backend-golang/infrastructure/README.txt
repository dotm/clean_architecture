You can put 3rd party libraries that interacts with external API here.

Some example:
- data storage library
    sql library to store data in a RDBMS (PostgreSQL, MySQL, etc.)
    caching library to store data in RAM/main memory (Redis, memcached, etc.)
    other libraries (ElasticSearch, etc.)
- logging library (Graylog, etc.)
- cURL library to interact with other services through http
    example services: payment service, banking API, Google Map library, etc.
- other libraries (SMTP email server API, push notification API)
